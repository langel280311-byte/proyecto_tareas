# 📌 RiwiFlow — Sistema de Gestión de Tareas Kanban

RiwiFlow es una aplicación web tipo Kanban desarrollada con:

- HTML5
- TailwindCSS
- JavaScript Vanilla
- json-server

El sistema permite:

✅ Inicio de sesión  
✅ Gestión de tareas  
✅ Roles de usuario (admin / coder)  
✅ CRUD de tareas  
✅ Control de permisos  
✅ Persistencia con json-server
